# Практическая работа студента Яндекс Практикума

## React-burger

### Описание:

Сайт для космической бургерной, где можно собрать на свой вкус и сразу оформить заказ.

## Для запуска

```sh
git clone
npm install
npm start
```

### Для тестов

```sh
Jest: npm run test
e2e: npm run cypress:open
```
